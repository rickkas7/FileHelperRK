◆ sectors

size_t FileHelperRK::Usage::sectors = 0

Number of sectors used.

Each sector is 512 bytes. Each file takes one sector for metadata plus enough sectors to hold all of the data. Each directory takes one sector.

This is an approximate value and may vary from the actual usage.
