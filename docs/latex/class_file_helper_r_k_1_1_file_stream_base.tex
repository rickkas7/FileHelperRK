\doxysubsection{Detailed Description}
Class for providing Stream and Print object for reading or writing a file. 

Important limitations\+:
\begin{DoxyItemize}
\item This class is only for reading or writing, not both at the same time
\item When reading from a stream, the underling file is not expected to change; it cannot be used for live tail for example 
\end{DoxyItemize}
