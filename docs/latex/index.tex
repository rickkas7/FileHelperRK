 {\itshape Useful functions for working with the Particle flash file system}

This library contains a number of utility functions for working with the Particle flash file system on Gen 3 and Gen 4 devices, such as\+:


\begin{DoxyItemize}
\item Store or read a string in a file
\item Store or read an array of bytes in a file
\item Store or read a Variant in a file. This can be used for structured data, including JSON (Device OS 5.\+6.\+0 and later).
\item Create a directory and parent directories (mkdirs)
\item Delete a directory recursively (delete\+Recursive)
\item Measure disk usage of a directory
\item Walk the directory tree and call a callback or labmda for each file or directory
\item Parse a pathname
\item Join pathname components
\end{DoxyItemize}
